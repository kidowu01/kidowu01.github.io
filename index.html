SARS Antiviral Bioactivity Prediction

AI-driven pipeline for prioritizing antiviral compounds

This project predicts IC50 values for SARS coronavirus 3C-like proteinase inhibitors using ChEMBL data and a Random Forest model (R² = 0.57). By computing descriptors like Molecular Weight and LogP with RDKit, it prioritizes drugs like lopinavir, saving millions in R&D costs. Enhanced with scatter plots and feature importance visualizations for interpretability. View on GitHub.

High-throughput screening for novel antivirals

This platform accelerates antiviral drug discovery by screening thousands of compounds for activity against viral targets like SARS-CoV-2.
